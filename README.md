# 🌌 Knowledge Cosmos v4

> 自分の知識に問いかける宇宙

**Knowledge Cosmos は読書管理アプリではありません。**  
読んだ本を記録することが目的ではなく、  
「自分が人生を通じて問い続けてきたこと」を可視化するためのプロダクトです。

---

## 課題

何冊読んでも、その知識が人生に活きている実感がない。

- この考え方はどの本だったか思い出せない
- 本同士のつながりが見えない
- 今の悩みと過去の読書体験が結びつかない

結果として、知識が点のままになり、人生の問いに活かされない。

---

## コンセプト

Knowledge Cosmos は **「自分の知識に問いかける宇宙」** である。

- AIは答えを出さない
- AIはユーザー自身の知識を再接続し、問いを返す
- SNSのように他人を見る場所ではなく、自分自身と対話する場所

---

## 機能一覧

| 機能 | 説明 |
|------|------|
| 📚 本棚写真アップロード | スマホで本棚を撮影 → AIが背表紙をOCR解析 |
| ✦ 宇宙への反映 | 本が光る星として表示。キーワード一致で線がつながる |
| 🤔 自動メタデータ生成 | タイトルからキーワード・テーマ・問いをAIが生成 |
| ☄️ 悩み入力 | 今の悩みを入力すると、関連する本の星が発光する |
| 💜 なぜ光ったか表示 | 本とあなたの悩みの接点を説明し、問いを返す |
| 🚫 AIは答えない | 「転職すべき」ではなく「何を失うことを恐れていますか？」 |

## 使用技術

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **AI**: OpenAI API (gpt-4o / gpt-4o-mini)
- **Hosting**: Vercel

## セットアップ

```bash
# リポジトリのクローン
git clone https://github.com/chisato-flfl/knowledge-cosmos-v4.git
cd knowledge-cosmos-v4

# 依存関係のインストール
npm install

# 環境変数の設定（任意）
cp .env.local.example .env.local
# OPENAI_API_KEY を設定するとフル機能が使えます
# 設定しない場合はデモモードで動作します

# 開発サーバー起動
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## デモモードについて

OpenAI APIキーなしでも動作します。

- 本棚写真アップロード → サンプルの本が追加されます
- 悩み入力 → キーワードマッチングで関連本を発見
- AIからの問いはあらかじめ用意されたものが表示されます

APIキーを設定すると：
- 実際の本棚写真からタイトルをOCR抽出
- 任意の本のメタデータをAIが生成
- 悩みと本を意味的に接続
