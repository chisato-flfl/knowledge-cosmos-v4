# 🌌 Knowledge Cosmos v5

> 自分の知識に問いかける宇宙

**Knowledge Cosmos は読書管理アプリではありません。**  
ユーザー自身の読書体験・人生経験・問いの蓄積を可視化するための「知識宇宙」です。  
目的は答えを出すことではなく、**本と人生を往復しながら、次の問いを発見すること**を支援します。

---

## コンセプト

- SNSのように他人を見るサービスではなく、「自分の知識に問いかける」場所
- AIは答えを出さない。問いを深める
- 知識の点が、問いによって線になる

---

## 機能一覧

| 機能 | 説明 |
|------|------|
| 📚 本棚写真アップロード | スマホで本棚を撮影 → AIが背表紙をOCR解析 |
| ✦ 宇宙への反映 | 本が光る星として表示。キーワード一致で線がつながる |
| 🤔 自動メタデータ生成 | タイトルからキーワード・テーマ・問いをAIが生成 |
| ☄️ 悩み入力 | 今の悩みを入力すると、関連する本の星が発光・中央に集まる |
| 💜 共鳴した本 | 本とあなたの問いの接点を説明し、書物ごとの問いを返す |
| ✦ 宇宙から返ってきた問い | **v5新機能** — AIが「次に考えると面白そうな問い」を3〜5個提案 |
| 🚫 AIは答えない | 「転職すべき」ではなく「何を失うことを恐れているのだろう？」 |

---

## v5 新機能：問いの推薦システム

問いを入力すると、AIが **「次の問い」** を提案します。

```
ユーザー入力：「リベラルアーツ」

→ 関連する本が発光・中央に集まる

→ 宇宙から返ってきた問い：
  ・なぜ私は学び続けたいのだろう？
  ・知識と自由はどう関係しているのだろう？
  ・自分らしい生き方とは何だろう？
  ・何のために知識を得るのだろう？
```

**AIの振る舞い**
- 禁止：人生相談の回答 / 問題解決 / アドバイス
- 推奨：問い返し / 哲学的な視点 / 自己探究 / 内省

## 使用技術

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **AI**: Anthropic Claude API (`claude-opus-4-7`)
- **Hosting**: Vercel

## セットアップ

```bash
# リポジトリのクローン
git clone https://github.com/chisato-flfl/knowledge-cosmos-v4.git
cd knowledge-cosmos-v4

# 依存関係のインストール
npm install

# 環境変数の設定（任意）
cp .env.local.example .env.local
# ANTHROPIC_API_KEY を設定するとフル機能が使えます

# 開発サーバー起動
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## デモモード（APIキーなし）

APIキーなしでも全機能が動作します。

| 機能 | APIキーなし | APIキーあり |
|------|------------|------------|
| 本棚OCR | サンプル本が追加 | 実際の本棚を解析 |
| メタデータ生成 | 26冊のモックデータ | 任意の本を即時生成 |
| 関連本検索 | キーワードマッチング | 意味的な接続 |
| **宇宙からの問い** | **ルールベース生成** | **Claude が哲学的に生成** |

---

## アーキテクチャ（問い推薦システム）

```
POST /api/recommend-questions
  ├── ANTHROPIC_API_KEY あり → Claude claude-opus-4-7
  │     システムプロンプト：「問い返しのみ。答えは出さない」
  └── APIキーなし → lib/questionRecommender.ts
        キーワードマップ（約30カテゴリ×5問）でルールベース生成
```
